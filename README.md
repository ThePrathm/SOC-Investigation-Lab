# SOC Investigation & Digital Forensics Lab

## 📌 Project Overview
This project simulates a real-world SOC (Security Operations Center) investigation scenario involving suspicious login activity, brute-force attempts, and potential unauthorized access.

The objective was to analyze logs, identify Indicators of Compromise (IOCs), and perform basic incident response.

---

## 🧠 Key Objectives
- Analyze Windows & web server logs
- Detect suspicious login patterns
- Identify brute-force attack behavior
- Perform log correlation
- Generate incident response report

---

## 🛠️ Tools Used
- Splunk (Log Analysis)
- Wireshark
- Nmap
- Windows Event Logs

---

## 🔍 Investigation Steps

1. Collected and analyzed 500+ log entries  
2. Identified multiple failed login attempts  
3. Detected brute-force attack pattern  
4. Correlated logs to identify suspicious IP addresses  
5. Extracted Indicators of Compromise (IOCs)  
6. Created incident response report  

---

## 🚨 Key Findings
- Multiple failed login attempts from same IP  
- Suspicious login behavior indicating brute-force attack  
- Unauthorized access attempts detected  

---

## 📊 Indicators of Compromise (IOCs)
- Malicious IP addresses  
- Abnormal login attempts  
- Suspicious timestamps
